Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

Riddet

Casusbeschrijving:
Dit project is een nabootsing van de website Reddit. Het idee is dat gebruikers een account aan kunnen maken en vervolgens hun eigen community op kunnen zetten of wellicht aan die van andere te participeren. Binnen deze communities kunnen verschillende threads aangemaakt worden waar andere gebruikers over kunnen discussiëren. Dit gebeurd door middel van het plaatsen van reacties. Op deze reacties kunnen meerdere reacties worden geplaatst. Naast deel te nemen aan communities en het lezen van threads is het ook mogelijk om profielen van andere gebruikers te zien en ze te volgen.

Relatiebeschrijving tussen entiteiten:
  • User
    • Een gebruiker staat op zichzelf en is dan ook niet afhankelijk van andere entiteiten.
    • Een gebruiker kan andere gebruikers volgen.
    • Een gebruiker kan reacties plaatsen op threads en onderliggende berichten.
    • Een gebruiker kan zelf een community aanmaken maar ook lid worden van andere communities.
    • Een gebruiker kan een thread aanmaken in een community waar hij lid van is.
  • Community
    • Een community wordt aangemaakt door een gebruiker.
    • Er kunnen meerdere gebruikers deelnemen aan een community.
  • Thread
    • Een thread is een onderdeel van een community. Binnen een community kunnen er meerdere threads aangemaakt worden.
  • Message
    • Is gelinkt aan een thread, een message is dus ook een bericht op een thread.
    • Messages kunnen ook weer messages bevatten. Dit kun je zien als reacties.
  • Category
    • Kan worden gebruikt om aan te geven welke categorieën een community of thread bevat.

Entiteiten

User (strong entity)

  • _id: ObjectId
  • username: string
  • firstName: string
  • lastName: string
  • dateOfBirth: Date
  • emailAddress: string
  • password: string
  • userImageUrl: string
  • creationDate: Date
  • isActive: boolean

Community (weak entity)

  • _id: ObjectId
  • name: string
  • description: string
  • creationDate: Date
  • imageUrl: string
  • isPublic: boolean

Thread (weak entity)

  • _id: ObjectId
  • title: string
  • publicationDate: Date
  • imageUrl: string
  • externLink: string
  • upvotes: Number

Message (weak entity)

  • _id: ObjectId
  • text: string
  • likes: Number
  • dislikes: Date
  • publicationDate: Date

Category (strong entity)

  • _id: ObjectId
  • categoryName: string

User stories

  • Als gebruiker wil ik kunnen inloggen, zodat ik gebruik kan maken van de applicatie
  • Als gebruiker wil ik een account aan kunnen maken, zodat ik toegang heb binnen de applicatie.
  • Als gebruiker wil ik een community aan kunnen maken, zodat ik mijn passie met meerdere mensen kan delen d.m.v. threads
  • Als gebruiker wil ik een categorie mee kunnen geven aan een community, zodat mensen die deel willen nemen aan mijn community, weten waar het over gaat.
  • Als eigenaar wil ik de informatie van mijn community kunnen wijzigen, zodat ik foutieve data kan veranderen.
  • Als eigenaar wil ik mijn community kunnen verwijderen, zodat ik afstand kan doen van ongewilde communities.
  • Als eigenaar wil ik gebruikers uit mijn community kunnen verwijderen, zodat ik afstand kan doen van ongewilde gebruikers.
  • Als eigenaar wil ik threads uit mijn community kunnen verwijderen, zodat er geen ongepaste threads worden gepost
  • Als gebruiker wil ik een thread aan kunnen maken binnen een community, zodat ik een gespreksonderwerp voor kan stellen binnen communityleden.
  • Als gebruiker wil ik de informatie van mijn aangemaakte thread kunnen wijzigen, zodat ik foutieve data kan wijzigen.
  • Als gebruiker wil ik mijn aangemaakte thread kunnen verwijderen, zodat ik hier geen omkijk meer naar heb.
  • Als gebruiker wil ik binnen een thread comments kunnen plaats, zodat ik mijn mening kan uiten over bepaalde threads.
  • Als gebruiker wil ik de geplaatste comments kunnen wijzigen, zodat ik eventuele spelfouten kan verbeteren.
  • Als gebruiker wil ik geplaatste comments kunnen verwijderen, zodat ik foutgeplaatste comments weg kan halen.
  • Als gebruiker wil ik bij het aanmaken van een thread een categorie mee kunnen geven, op deze manier kunnen mensen een duidelijker beeld krijgen van waar de thread over gaat.
  • Als gebruiker wil ik kunnen reageren op comments, zodat meningen kunnen worden geuit over de comments.
  • Als gebruiker wil ik deze geplaatste reacties kunnen wijzigen, zodat ik eventuele spelfouten kan verbeteren.
  • Als gebruiker wil ik deze geplaatste reacties kunnen verwijderen, zodat ik foutgeplaatste reacties weg kan halen.
  • Als gebruiker wil ik deel kunnen nemen aan een community, zodat ik comment binnen threads in die groep kan plaatsen.
  • Als gebruiker wil ik een community kunnen verlaten, zodat ik geen onderdeel meer ben van deze community.
  • Als gebruiker wil ik andere gebruikers kunnen volgen, zodat ik ze in de gaten kan houden.
  • Als gebruiker wil ik andere gebruikers kunnen ontvolgen, zodat ik ze niet meer in de gaten ga houden.
  • Als gebruiker wil ik mijn profiel kunnen zien, zodat ik een indruk heb van hoe andere mensen mijn profiel zullen zien.
  • Als gebruiker wil ik de data van mijn profiel kunnen aanpassen, zodat ik incorrecte data kan verwijderen.
  • Als gebruiker wil ik mijn account kunnen verwijderen, zodat mijn gegevens niet meer bekend zijn binnen de applicatie.
  • Als gebruiker wil ik het profiel van andere gebruikers kunnen bekijken, zodat ik informatie over hen kan zien en ze eventueel kan gaan volgen.
  • Als gebruiker wil ik een overzicht hebben van alle thread oplopend gesorteerd, zodat ik door deze heen kan scrollen.
  • Als gebruiker wil ik een overzicht hebben van een community pagina, zodat ik hier informatie over kan lezen.
  • Als gebruiker wil ik op het community overzicht een overzicht hebben van alle threads, zodat ik kan zien wat voor threads er allemaal zijn binnen de server.
  • Als gebruiker wil ik een detailpagina van een thread kunnen zien met hierin de comments, zodat ik een duidelijker beeld heb van de thread en hierop kan reageren.
  • Als gebruiker wil ik kunnen zoeken naar communities, zodat ik de details kan bekijken en eventueel deel kan nemen aan deze communities.
  • Als gebruiker wil ik kunnen zoeken naar andere gebruikers, zodat ik makkelijk mijn vrienden kan vinden.
  • Als gebruiker wil ik een overzicht hebben van communities waar ik nog niet aan heb deelgenomen, zodat ik een overzicht heb van communities waar ik nog aan kan deelnemen.
  • Als gebruiker wil ik een overzicht hebben van communities waar ik aan deelneem, zodat ik snel kan schakelen.
  • Als gebruiker wil ik op de homepagina filters hebben van bijv.: categorie, community, aantal reacties, datum, zodat ik specifiekere threads kan bekijken.
  • Als gebruiker wil ik op het community overzicht filters hebben van bijv.: categorie, community, aantal reacties, datum, zodat ik specifiekere threads kan bekijken.
  • Als gebruiker wil ik filters hebben op overzicht van niet deelgenomen communties van bijv.: categorie, zodat ik specifiekere threads kan bekijken.
  • Als gebruiker wil ik binnen een community kunnen filteren op mijn eigen gemaakte threads, zodat ik niet lang hoef te zoeken om deze te vinden.

User stories (aanbevelingen/complexe queries)

  • Als gebruiker wil ik aanbevelingen krijgen van mogelijke communities die ik kan joinen, zodat ik zonder zelf te zoeken leuke communities tegen kan komen
    • Hiervoor moet er voor alle mensen die ik volg worden gekeken of er meerdere mensen zijn die onderdeel uitmaken van dezelfde community. Dit kan bijvoorbeeld worden gedaan door het op community te groupen en het hoogste aantal aan te bevelen of de top 3 etc.
  • Als gebruiker wil ik aanbevelingen krijgen van gebruikers die worden gevolgd door mensen die ik volg, zodat ik nieuwe mogelijk interessante mensen in de gaten kan houden.
    • Hiervoor moet er voor alle mensen die ik volg gekeken worden of deze overeenkomende gebruikers volgen. Net als de bovenstaande query kan er dan weer gekeken welke het meest gevolgd wordt.
  • Als gebruiker wil ik aanbevelingen krijgen van leuke threads waar meerdere van de mensen die ik volg op hebben gereageerd, zodat ik mee kan discussiëren met de mensen die ik interessant vind.
    • Hiervoor moet er voor alle mensen die ik volg gekeken worden als er reacties plaatvinden binnen verschillende threads.
